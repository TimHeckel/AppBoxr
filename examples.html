How do these Examples even work? AppBoxr as a proxy!

Because I'm hosting these pages on Github, they're all vanilla HTML. BUT, I pushed a new Install-Package AppBoxr -Pre solution up to AppHarbor and have changed my AppBoxr objects to use https://appboxr.apphb.com.

Now, I have pure javascript / html / css examples doing all the AppBoxr goodness, yet using a remote server. This cross-domain kungfu is all due to SignalR.
